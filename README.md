# USA Regional Sales Analysis — Exploratory Data Analysis (EDA)

**Author:** Peddula B Kavya

## Overview
This project applies exploratory data analytics to a USA regional sales dataset to derive insights that can help improve sales performance and drive business strategies. It analyzes sales across 6 product lines, 4 sales channels, and 8 U.S. regions.

## Contents
- `EDA_Regional_Sales_Analysis.ipynb` — full exploratory analysis (Python: Pandas, Matplotlib, Seaborn)
- `Regional Sales Dataset.xlsx` — source dataset
- `Sales_data(EDA Exported).csv` — cleaned/feature-engineered dataset exported from the EDA
- `site/` — static landing dashboard (HTML + Chart.js) summarizing the key findings

## Key Findings
- The top ~10% of products contribute over 60% of total revenue and 55% of profit margin.
- California leads regional performance (~$230M revenue, 7,500+ orders).
- Channel mix: Wholesale 54%, Distributors 31%, Exports 15%.
- Over 200 high-risk outlier transactions flagged for pricing anomalies or margins below 5%.

## Tools
Python (Pandas, Matplotlib, Seaborn), Jupyter Notebook.
